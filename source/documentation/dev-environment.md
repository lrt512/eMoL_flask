# Development Environment

## Get Started
* Install [Vagrant](https://www.vagrantup.com/)
* Install [VirtualBox](https://www.virtualbox.org/)
* Open shell in ```emol/vm```
* ```vagrant up```

## Edit Code
It's in ```emol/source```

## Run Some Unit Tests
* Open shell in ```emol/vm```
* ```vagrant ssh```
* ```cd ~/source/emol```
* ```py.test```
